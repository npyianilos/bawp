# Apricot React Components - Documentation Index

## Usage for AI Agents

**Start here first.** This is a quick reference index of available Apricot components and CSS utilities. Only read the detailed documentation file for the specific component or utility you need to work with - do not load all files at once.

React components are generally imported like this: import { ComponentName } from "@cb/apricot-react"

## React Components

- **Button** - Primary, secondary, tertiary, link, and naked button variants with sizing and states. See [button.md](button.md)
- **Cards** - SimpleCard, EventCard, MediaCard, ResourceCard, CTACard, and PromoCard for displaying modular content. See [cards.md](cards.md)
- **Checkbox** - Checkbox and CheckboxGroup components with controlled/uncontrolled modes and indeterminate state. See [checkbox.md](checkbox.md)
- **DatePicker** - Date selection component with calendar interface and validation. See [date-picker.md](date-picker.md)
- **Dropdowns** - Dropdown, DropdownMenu, and DropdownItem components for select menus and actions. See [dropdowns.md](dropdowns.md)
- **Icon** - Icon component with 200+ icons from College Board icon library. See [icon.md](icon.md)
- **Input** - Text input component with validation, character counter, and various input types. See [input.md](input.md)
- **InputDropdown** - Combo input with integrated dropdown for autocomplete and search. See [inputdropdown.md](inputdropdown.md)
- **InputStepper** - Number input with increment/decrement buttons and min/max validation. See [input-stepper.md](input-stepper.md)
- **SelectDropdown** - Enhanced select with floating labels, grouped options, and validation. See [select-dropdown.md](select-dropdown.md)
- **Modal** - Modal dialog component with video support and customizable behavior. See [modal.md](modal.md)
- **Notifications** - Notification banner component for alerts, success, warnings, and info messages. See [notifications.md](notifications.md)
- **Pagination** - Pager, PageSize, and PaginationInfo components for table and list pagination. See [pagination.md](pagination.md)
- **RadioButton** - RadioButton and RadioButtonGroup for single-selection with horizontal/vertical layouts and validation. See [radio-button.md](radio-button.md)
- **Spinner** - Loading spinner with static, animated, and progress modes. See [spinner.md](spinner.md)
- **Table** - Table and Column components with sorting, pagination, loading states, and expandable rows. See [tables.md](tables.md)
- **Tabs** - Tabs, TabList, Tab, TabPanel components with keyboard navigation and controlled/uncontrolled modes. See [tabs.md](tabs.md)
- **TextArea** - Multi-line text input with floating labels, validation states, and flexible sizing. See [textarea.md](textarea.md)
- **ToggleSwitch** - On/off toggle switch with labels, description, and controlled/uncontrolled modes. See [toggle-switch.md](toggle-switch.md)

## CSS Utilities

- **Borders** - Border width, color, style, radius, and box shadow utility classes. See [borders.md](borders.md)
- **Colors** - Color palettes (CB, K-12, Higher Ed, Neutrals, System) with background, gradient, and opacity classes. See [colors.md](colors.md)
- **Grid** - 12-column responsive grid system with flexbox utilities and breakpoints (xs-3xl). See [grid.md](grid.md)
- **Spacing** - Margin and padding utility classes with responsive variants. See [spacing.md](spacing.md)
- **Typography** - Font families, sizes, weights, colors, headings, paragraph styles, and text utilities. See [typography.md](typography.md)
